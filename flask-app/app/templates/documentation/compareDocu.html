Compare Algorithms

Here we check for all receivers and all defined algorithms whether the signal reaches a certain minimum value or not and we sum up all which reach this minimum value. In addition, we sum up the used amplitude factor 'a' of the senders for each algorithm, which can be seen in the top legend of the plot. We compare all defined and all working algorithms. The positions of the senders and receivers are the same for all algorithms.

Number

You can set the random seed which influences the position of the senders and receivers, so that senders and receivers always have the same position with the same number of senders, receivers and same random seed. The required calculation factor is shown under Log10(calculation) and cannot take a value above 9.

Receiver Positions

With Formation you can choose in which form the receivers are distributed, 3d sphere, 2d horizontal disc, 2d vertical disc. X,Y,Z are the coordinates of the origin of the choosen shape and Radius the corresponding radius in which the receiver are randomly distributed.

Parameters

With isotropic you can decide whether a sender sends equally in all directions or not. See Isotropic radiator, Dipole antenna

Path loss, Wavelength and b are parameters of the model. The parameters Wavelength and b are passed to the algorithms.

Plot Settings

Min signal value gives the minimum signal value at which a receiver must be reached in order to be counted.

With Signal you can choose which part of the received signal must exceed the minimum signal value. For example, only the imaginary part or only the real one.

Plot Range specifies the range in which is added to the minimum signal value and within which it is checked whether the receivers reach the minimum.

You can set the ignore receiver in origin radius so that all receivers that are within the selected radius from the origin (0,0,0) are not counted in the plot.

Simulate

Each algorithm is simulated and then the plot is created. If only values ​​under plot sessings are changed, the algorithms are not re-simulated but the changes are applied to the plot.
