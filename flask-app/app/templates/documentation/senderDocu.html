Sender

In this plot, each point corresponds to a sender that is placed randomly within the radius 1 disc. All senders send to a the same receiver. The signal at the receiver results from the addition of the received signals: model. The signals amplify or cancel each other out according to the superpositions principle.

Number

You can set the random seed which influences the position of the senders, so that with the same number of senders and same random seed the senders always have the same position.

Receiver Positions

X,Y,Z are the coordinates of the receiver.

Parameters

With isotropic you can decide whether a sender sends equally in all directions or not. See Isotropic radiator, Dipole antenna

Path loss, Wavelength and b are parameters of the model. The parameters Wavelength and b are passed to the algorithms. You can select which algorithm is used.
