Receiver Sphere

In this plot, each point corresponds to a receiver that is placed randomly within a sphere. All receiver receives signals from the same Senders. The Senders are placed randomly within a radius 1 disc. The signal at the receiver results from the addition of the received signals: model. The signals amplify or cancel each other out according to the superpositions principle.

Number

You can set the random seed which influences the position of the senders and receivers, so that senders and receivers always have the same position with the same number of senders, receivers and same random seed. The required calculation factor is shown under Log10(calculation) and cannot take a value above 9.

Receiver Positions

X,Y,Z are the coordinates of the origin of the choosen shape and Radius the corresponding radius in which the receiver are randomly distributed.

Parameters

With isotropic you can decide whether a sender sends equally in all directions or not. See Isotropic radiator, Dipole antenna

Path loss, Wavelength and b are parameters of the model. The parameters Wavelength and b are passed to the algorithms. You can select which algorithm is used.

Plot Settings

Cut origin radius cuts that all receivers that are within the selected radius from the origin (0,0,0) in the plot.

Min and Max are bounds, so that all values ​​outside the cabinet are set to the respective minimum or maximum.

With Plot you can choose which part of the received signal is displayed. For example, only the imaginary part or only the real one.

You can set the colormap and change with Elevation and Azimuth the angles of the Plot.

Simulate

If only values ​​under plot sessings are changed, the algorithms are not re-simulated but the changes are applied to the plot.
