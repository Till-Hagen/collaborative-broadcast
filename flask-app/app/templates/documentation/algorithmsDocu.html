Add/Edit/Delete Algorithm

Here you can add, edit or delete algorithms. The defined algorithms are stored as a string in a Sql database and compiled at runtime. New modules can also be imported and the operating system inside the container can be accessed. Due to its power, this feature is only available for local use.

Add

You can add new algorithms by selecting "new", specifying a name, filling in the method and press save. The return value must be a 2-tuple, the first and the second value of the tuple must be type of ndarray with size ​​equal to the number of senders. The first value determines the phase shift, the second the amplitude factor.

For example: The Supernova algorithm

import cmath
import math
import numpy as np

def new( _X: ndarray, _Y: ndarray, _wavelength: float, _b: float):

    distanceToOrigin = np.sqrt(np.square(_X) + np.square(_Y))
    cons = 2 * cmath.pi / _wavelength
    varphi = _b * cons * distanceToOrigin
    a = np.ones(_X.shape)
    return (varphi, a)

Pressing Save, the method is tested and if there is a syntactic or semantic error the corresponding error message appears. If the test runs through, the defined algorithm can be used in all modules.

Edit

If you select an algorithm, you can rename it or change the behavior in the method. The change is saved by pressing save. If the test does not run through, the algorithm is marked accordingly and cannot be used in other modules.

Delete

If you delete a selected algorithm, it is permanent, it cannot be restored. It can no longer be accessed in other modules.
